Projects

Path finding visualizer

This project is a collection of path searching algorithms visualized. So far you can make paths (place/destroy walls), move the goal/start, generate mazes, and do the searches. The searching algorithms that I have implemented is A*, Dijkstra, Breadth First, Depth First, and Greedy First Searches. The maze algorithms so far is recursive/iterative backtracking and Hunt and Kill.

Sub Sinker

Sub Sinker is a multiplayer stealth shooter designed in Unity. Using techniques such as Cellular Automata and Marching Squares, every environment/map is randomly generated, allowing the players to show their skills in stealth and perception rather than memorization. The game is also networked, meaning it is played across multiple different devices while still not putting too much load on the main host.

osu! Scraper

osu! Scraper is a python web scraper for the game osu! Utilizing Beautiful Soup, the web scraper is able to find player profiles and display them in readable formats. It can also ouput JSON files that can be used for various other activities. GUI is partially implemented and multithreading is an implemented feature (though not used due to overflowing web requests on osu!'s side)

Cross Contamination

Cross Contamination is a family friendly FPS created with Unity. With creative food-based enemies like rolling Oreos, flying Donuts, or even slithering Mape Bars, Cross Contamination creates an immersive environment to survive against the odds.
